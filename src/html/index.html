What Is This Site?

Coding for Coconuts is a website developed by Morgan Fleshren, Alexa Grau, and Katherine Sanchez as part of our Software Engineering class project. It is a fun, free, and interactive website for middle schoolers (age range 9-13) to begin learning coding. This includes beginner coding logic as well as some simple coding examples. To keep the users interested, this website contains fun tasks such as problems that are interactive yet challenging. As the user progresses through each level, they will collect coconuts. Harder levels are worth more coconuts to reward the user accordingly. This project will benefit students, parents, and teachers as the users develop a basic understanding of programming through this website for free.

Who Are You?

This website is ideal for anybody wanting to dip their toes into the world of coding. This website provides a basic introduction to coding logic as well as some coding examples. Our targeted age group is 9-13 years old, however, this website is useful for anybody beginning their coding journey.

How to Begin?

Click the 'Questions' tab in the top right to begin learning, or click here. To keep track of your coconut collection progress, head over to the 'Coconut Counter' tab, or click here. If you get stuck, feel free to use the 'Solutions' tab to take a sneak peek at the answers, or click here. Make sure to give each problem your best effort before looking at the questions, though! You'll only gain as much as you put in.
